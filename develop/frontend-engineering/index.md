## Principle

Codifies the framework patterns that keep web UIs fast as they grow. Use
when writing, reviewing, or refactoring component, data-fetching, or
rendering code — especially in React/Next.js.

## Choosing a variant

Today there's one variant — Vercel's `react-best-practices` skill. Reach
for it when you're on React or Next.js and want a single rulebook from the
team that maintains the framework.
